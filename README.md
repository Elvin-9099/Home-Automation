# Home Automation
This project develops a voice recognized smart home automation system.
## System Requirements
### 1. Hardware Requirements
       1. NodeMCU with ESP8266 Wi-Fi module
       2. Relay Module
       3. Electrical Appliances
       4. Sensors (optional)
### 2. Software Requirments
       1. Arduino IDE
       2. Adafruit IO 
       3. If This Then That (IFTTT)
